Overview

Learn how to structure, align, and create responsive layouts using Flexbox and Grid. Apply Responsive Web Design principles to make your page adapt across all devices.

Objective

Build a responsive webpage with Flexbox and Grid that looks good on desktop, tablet, and mobile.

What You’ll Practice

Using Flexbox for 1D layouts
Using Grid for 2D layouts
Responsive design with media queries
Adapting layouts across screen sizes

Instructions

  1. Create a header, navigation, content area, sidebar, and footer.
  2. Use Flexbox in one part and Grid in another.
  3. Apply media queries for responsiveness.

Deliverables

  • index.html – HTML5 structure
  • style.css – responsive CSS with Flexbox, Grid, and media queries

Tips

  • Use both Flexbox and Grid effectively.
  • Keep the layout clean and responsive.
  • Use comments in your CSS.
  • Be creative with your layout.
